Calculates pruned path, builds perpendiculars to the original path.

This finds the path segmen s in the original path path with the shortests distance to point, measured perpendicular to s. The path segment s is defined as the straight line from waypoint s to waypoint s+1. The pruned path starts with point and continues with all waypoints in path starting with waypoint s+1.

If two or more segments in path have the same highest distance to point, then the segment which comes last in path is selected.

Parameters
[in]path- std::vector<FrameWithId> original path, a vector of FrameWithId.
[in]point- FrameWithId trimming point.
[out]prunedPath- std::vector<FrameWithId> resulting pruned path.
