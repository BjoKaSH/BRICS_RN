Demo application

The BRIC_3D_Demo program demonstrates some core capabilities of the BRICS_3D library. It will start by loading either a depth image or a point cloud, down-sample the data in an Octree-based filtering step, optionally register two point clouds into common coordinate system, creates triangle mesh and visualize the result.

Simple point cloud viewer

Simple point cloud viewer program loads a point cloud from a txt file, optionally filters the data with an Octree based sub-sampling filter to reduce the data and finally visualizes the result.
