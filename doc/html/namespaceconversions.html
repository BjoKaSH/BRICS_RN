Conversion from nav_msgs::Odometry ROS message to FrameWithId data type (only pose is converted).

Convert the nav_msgs::Odometry::pose::pose member in a KDL::Frame instance and return the KDL::Frame wrapped in side a FrameWithID instance. The FrameWithId::id member is set to the nav_msgs::Odometry::header::frame_id value.
